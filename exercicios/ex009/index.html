Outras formatações

Codigos

Para códigos podemos usar a TAG code que é uma tag para codigos, porém, ela não vem com formatação. Ex;

num = int(input("código do número") if num % 2** @ print(f"o" numero {num} é par) else: print(f"o número {num} é impar) print("fim do programa").

E tamém podemos usar a tag code dentro de uma TAg chamada PRE que é uma tag que tem formatação. EX

 
        num = int(input("código do número")
        if num % 2** @
        print(f"o" numero {num} é par)
        else:
        print(f"o número {num} é impar)
        print("fim do programa").

Citações completas

Para citações completas podemos usar a tag blockquote para fazer referencia ao autor. Ex:Como diz o maior heroi do mundo em boku no hero

Um herói de verdade é alguém que supera as adversidades da vida.

Abreviações

Posso usar a tag ABBR para abreviações. Ela ira mostrar uma pequena caixa de popup explicando o que é significa as minhas abreviações. EX: Estou adorando estudar HTML E CSS
